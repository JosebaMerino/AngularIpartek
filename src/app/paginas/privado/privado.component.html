Zona privada

Solo pueden acceder usuarios logeados

Para proteger el acceso a esta ruta debemos usar una guarda (GUARD) y definirla en routing.

path: 'privado', component: PrivadoComponent, canActivate: [LoginGuard]
canActivate
Guarda para indicar si se puede activar esa ruta
LoginGuard
Guarda creada por nosotros que retornara true o false
